# Data Directory

This directory is intentionally excluded from version control and is used for storing external data files required to run the notebook.

Expected structure:

data/
└── Companies-AI-Initiatives/
    ├── company_a_ai_strategy.pdf
    ├── company_b_innovation_report.pdf
    └── company_c_ai_overview.pdf

To run the project:
1. Create the required folder structure locally or in Colab.
2. Place relevant AI-related corporate PDF documents inside the directory.
3. Update the path in the notebook if necessary.

Note: Data files are not included in this repository.
